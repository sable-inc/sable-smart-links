Usage Instructions

To use these functions in your own project:

import { highlightElement, restoreElementStyle } from "sable-smart-links/interactor";

// Highlight an element
const element = document.getElementById('my-element');
const styleInfo = highlightElement(element);

// Later, restore the original style
restoreElementStyle(element, styleInfo);
